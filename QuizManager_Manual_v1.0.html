  1. The Quiz Manager

    This section describes how to use Quiz Manager to take quizzes. You will find here a step-by-step guide about how to select a quiz in your file system, choose a quiz mode and how it will behave while your are taking a quiz.
    You will also find what all the items in menu bar at the top of the window do.

    So now, without any further delay, let's get you started with Quiz Manager!

    1. Taking your first quiz

      Here are the few steps you need to follow to launch a quiz thanks to Quiz Manager.

      1. Launching Quiz Manager

        It could seem obvious but well, you need to launch the software first. Usually a double-click on QuizManager.jar wherever you placed it in your file system should be enough. If you haven't downloaded the program yet, you can do so by clicking the following link.
        The following window should now appear:

        If the window doesn't appear, maybe you haven't installed the latest version of Java Runtime Environment (or JRE for short). You can find instructions on how to install JRE on your computer here. If after installing JRE and resarting your computer you still don't manage to launch the program then you can try this. Best of luck!

      2. Opening a Deck on your file system

        You should now have window looking like this one.

        Now you may wonder what we mean by 'Deck'. A Deck is the word we use to describe a series of questions and their answers which Quiz Manager is able to handle. In Quiz Manager, each question contains the following information:

        • A question
        • A correct answer
        • Three wrong answers
        • An explanation

        If you want to create your own decks, I sugggest you directly skip to the Deck Editor section. If you would like to try Quiz Manager but you don't want to create a deck yourself, you can download one here.

        To open a deck file located in your files, you need to go into the 'File' menu located at the top left corner of the window. You then need to click on the item labeled as 'Load Deck'.

        As an alternative you can use the following keyboard shortcuts:

        1. Alt + F to open the 'File' Menu.
        2. L (equivalent to clicking on the 'Load Deck' item).

        A file chooser dialog now opens. It should look like this:

        You can now browse your file in search of a deck file. Those files end with a '.dk' extension. You can use the filter of the file chooser to choose to display either all your files or only the folders and the deck files to make the search easier.

        If you happen to find a dk file on your system that doesn't look like a deck file, it may be used by another program that happens to use the same extension to store data. It is probably safer to avoid trying to open those files with Quiz Manager! But if you like to live dangerously, then it is your problem ;).

      3. Choosing a Quiz mode

        This step should be pretty simple, it is about being able to click on a button on the screen. You are probably more interested in the difference between the different choices that appear on your screen:

        • Learning mode

          Learning mode is, true to its name, intented for learning purposes. In short, the idea is the following: Bis repetita placent.

          When you answer a question with the good answer, the buttons on which the possible answers were displayed switches to a congratulation message. You can then process to the next question by clicking the button just below.
          However, if you provide a wrong answer, the answer buttons will be replaced by the exaplanation. And you can then proceed to the next question by clicking on the button just below.

          And so you would think that you woul just go through all the questions until the deck runs out of question? Well ... not quite.

          The question will keep reappearing until you provide enough correct answers. This 'enough correct answers' amount can be changed by following these instructions.

        • Test mode

          The test mode is a bit more trivial than the Learning Mode.

          All the questions of the deck will be displayed only once. As soon as you answer a question, the next one is displayed, until the decks runs out of questions. Unlike the learning mode, no explanation is displayed between two questions. A message will then be displayed with your score indicated as follows:

          In the example above, 'XX/TT' means the user got XX good answers out of TT questions.

      4. During the Quiz

        You will encounter two kinds of display during a quiz when using Quiz Manager. The question display and the explanation display.

        • Question

          When a question is presented to the user, it is displayed like this:

          At the top you have the question. It here reads 'XXXXXXXXXXXXX'.
          Below are four buttons that each contain a possible answer to the question. The software is programmed such that the position of the good answer changes from a question to another. So always clicking on the same button is basically useless!

          And then just below you have a 'I don't know' button. Clicking on this button will have the same effect as providing a wrong answer. Hence if you click on this button in learning mode, it will display the explanation.

        • Explanation (specific to the learning mode)

          You will only see the explanation view when running a quiz in learning mode.
          For more detailed information, you can refer to this section.

          When taking a quiz in learning mode, after each answer you provide, you will end up with a display like this one:

          You can see the question you just answered to is still displayed. But just below it, the answer buttons have been replaced by a text area. The content of this text area will vary, depending on if you provided a right or a wrong answer to the question.

          If you were right, the text area will contain a congratulation message.

          If you were wrong, the text area will contain the explanation associated with the question. Hopefully, the creator of the deck will have provided sufficient explanation for you to get a good answer next time!

    2. The menus in the MenuBar

      The menu bar located at the top of the window contains the following items:

      • File

        So far there is only one item in this menu: Load Deck. Its use is already detailed here.

      • Edit

        The Edit menu contains two items:

        • Create deck
        • Edit deck

        Both of these items will open a Quiz Editor window.

        The Create Deck item will directly launch a new Editor Window whereas the Edit deck will first open a file chooser dialog to allow you to open an existing deck file into the editor.

      • Preferences

        The preference menu contain a single item: Options. By clicking on this item you will open a dialog asking for the 'Number of good answers to pass a question in Learning mode'. The current value is then indicated in a text field.

        As indicated, this dialog lets you choose how many times you need to provide a good answer to a question for it not to appear again in learning mode. For more details about the way the learning mode works, you can go to learning mode section.

      • About

        The About menu contains two items: Credits and User Manual.

        • Clicking on Credit will open a new window containing various information about what you can and cannot do with this software. It also contains a copy of the GNU General Public License under which this software is published.

        • Clicking on User Manual will open this manual in your web browser.

  2. The Quiz Editor

    1. Creating your first Deck

      You will learn here the few but simple steps to follow to create a deck.

      1. Opening the Deck Editor

        First, you will need to open the Deck Editor. In the window that opens when you launch QuizManager.jar, click on the Edit menu (it is located next to the File menu). Two items now appear:

        Click on the Create Deck item. A new window should appear with the following appearance:

      2. Creating your questions

        You can see at the top of the window, your question number is displayed. As you are creating a new deck, it should display 'Card 1'. Below are 6 empty fields, each matching a Label.

        • The first field will contain the question. You can enter it by clicking on it and typing your question.

        • The second field will contain the correct answer. Try not to mess it up. There is no artificial intelligence checking that what you claim to be the answer to that question is really a good answer. If you screw it up, those who will then use the quiz are going to be confused!

        • The three following fields are meant to receive the wrong answers. Again try not to put a good answer in one of those field or you will confuse those who will take the quiz after you!

        • The last field is a bit larger and is meant to welcome the explanationto the question. Not that unlike all the other fields you can type on several lines. If you are typing a rather long explanation, do not hesitate to resize the window to display more text!

        And that's it! You have created your first question! Now you would ask:

        How do I create the next question?

        Well it is very simple. At the bottom of the window are two buttons: Prev and Next.
        Clicking on the Next button will open a new question form and you will be able to fill in the fields just like you just did for your first question! Notice that the card number indicated at the top of the window changes as you press the 'Next' button.

        If you want to change a question you wrote, click on the 'Prev' button. The text you entered will now appear again and you will be able to modify it.

        If you can't click on the 'Prev' button, it means you already are on the first question, and so, as there are no previous question to the first one, you can't go back.

        Warning

        If you do not fill in some of the fields in a question, the program will still work properly and you will be able to save your deck normally.

        But! When you will take a quiz using this deck with empty entries, nothing will be displayed where the empty field was empty in the editor. This might confuse the user. Hence, unless this is your goal, I recommend that you always fill in all the fields whan you create a question.

      3. Saving you Deck

        You have created a bunch of questions (or maybe just one) and you would like to save the deck you've just created in your file system. Doing so is very simple. Here is what you need to do.

        Click on the file menu in the menu bar at the top of the window. Then click on 'Save Deck'. A file chooser dialog like this one should now appear.

        You can now explore your folders to find a suitable location for your Deck. Maybe you want to create a new folder that will contain all of them, it's up to personal taste.

        Warning!

        Whichever name you give to your deck, please make sure you specified the extension of your file which should be .dk. If you don't save your deck with this format, you will not be able to open it, would you want to play the quiz or edit it.

        In the unlikely event where you were to find a file whose extension is .dk but you don't think is a quiz you created or downloaded, do not re-write it! It may be a file from another program, and erasing it may cause errors in this other software.

        As far as I know, no program uses the .dk extension but do not take this for granted! If you are aware or if you know a program uses this extension, please share the information by sending me an e-mail. Thank you!

        You can either select an existing file and re-write it. Be careful! You will forever loose the data contained in the file you selected!

    2. Editing an existing deck

      Here is how you can open an existing deck file to edit it.

      • From the Editor

        In the menu bar at the top of the window, click on the File menu and then on the 'Open Deck' item. This will open a file chooser dialog to let you choose which file you want to open. Once you have chosen the file you wish to open, the first question of the deck will appear in the window. You can now change the questions as explained above.

      • From the Player window

        In the menu bar at the top of the window, click on the Edit menu and then on the 'Edit Deck' item. This will open a file chooser dialog that will let you choose which deck you wish to edit. After you have chosen a file, a new Editor window will appear with the deck you selected. Even if you didn't choose any deck, the Editor window will appear but with empty fields.

        You can then follow the steps described above.

    3. The Editor Menu Bar items

      There are only two menus in the Editor. File and Edit.

      • The File menu

        Warning!

        Using any of the File menu item (exept the Save File item) will make you loose all the content you already have in the Editor. All the questions fields you were working on will be cleared without warning. Please make sure you saved what you want to keep before you press one of these buttons!

        • New Deck

          The 'New Deck' item will entirely clear all the fields of the current Deck you are working on. Beware, every question you typed which you haven't saved beforehand will be lost!

        • Open Deck

          The 'Open Deck' item will let you open an existing deck in the Editor. You will then be available to change the questions, add some etc. Beware! If you made changes to a deck but didn't save it, and open another deck, all those changes will be lost without warning! Make sure you saved your previous deck before opening an other one!

        • Save Deck

          The 'Save Deck' item will open a file chooser dialog to let you save the current content of the Editor. You can refer to the Save your Deck section above. Make sure you read the Warning, it will avoid you getting into trouble!

      • The Edit menu

        The Edit menu contains two items, Insert question and Remove question.

        • Insert question

          True to its name, the 'Insert question' item will let you insert a question where you are.

          More accurately, it will push the current question to the next position and create an empty question in the gap. If you press the 'Next' button, you will reach the question you were looking at just before.

        • Remove question

          True to its name, this item will remove the question you are currently looking at.

          If there exists a question after the one you just removed, this next question will be displayed. If not, that is you removed the last question of your deck, the previous (and new last) question will be displayed.

          Finally if your deck contains a single question, it will be cleared.

          Beware! The question you've just removed will be lost forever. You won't be able to find it again!

          Note: if your goal is to remove all the questions currently in your Editor, it will be much quicker to use the File > New Deck item!

      • About

        The About menu contains two items: Credits and User Manual.

        • Clicking on Credit will open a new window containing various information about what you can and cannot do with this software. It also contains a copy of the GNU General Public License under which this software is published.

        • Clicking on User Manual will open this manual in your web browser.
